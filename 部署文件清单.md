## 📦 打包上传建议

### 方法1: 创建压缩包(推荐)

在本地运行:
```bash
cd ~/Desktop/feishu_chajian_qianzi

# 创建部署压缩包(排除不需要的文件)
tar -czf backend-deploy.tar.gz \
  --exclude='BaseOpenSDK-Python-Playground/.venv' \
  --exclude='BaseOpenSDK-Python-Playground/__pycache__' \
  --exclude='BaseOpenSDK-Python-Playground/.git' \
  --exclude='BaseOpenSDK-Python-Playground/*.pyc' \
  --exclude='BaseOpenSDK-Python-Playground/poetry.lock' \
  --exclude='BaseOpenSDK-Python-Playground/pyproject.toml' \
  BaseOpenSDK-Python-Playground/

# 查看压缩包大小
ls -lh backend-deploy.tar.gz
```

然后上传 `backend-deploy.tar.gz` 到服务器,在宝塔面板解压。

### 方法2: 只上传必要文件

手动选择以下目录和文件上传:

## 🚀 宝塔部署步骤

### 1. 上传文件
- 上传压缩包或手动上传文件到 `/www/wwwroot/feishu_backend/`

### 2. 修改 .env 配置
```bash
APP_ID=你的飞书APP_ID
APP_SECRET=你的飞书APP_SECRET
HOST_URL=https://你的前端域名/
```

### 3. 在宝塔Python项目管理器中配置

| 配置项 | 值 |
|--------|-----|
| 项目路径 | `/www/wwwroot/feishu_backend` |
| Python版本 | Python 3.10+ |
| 框架 | Flask |
| 启动方式 | Gunicorn |
| 应用模块 | `main:app` |
| 端口 | `3000` |
| 依赖文件 | `requirements.txt` |

### 4. 安装依赖
- 点击"安装依赖"
- 选择 `requirements.txt`
- 等待安装完成

### 5. 启动项目
- 点击"启动"
- 查看日志确认启动成功

## 📊 文件大小参考

实际需要上传的文件很小:
- 代码文件: ~100KB
- 静态文件: ~50KB
- 总计: **< 1MB**

不需要上传虚拟环境,所以很快!

## ⚠️ 重要提醒

1. **不要上传 `.venv/` 目录** - 这个目录很大(几百MB),而且服务器会重新创建
2. **确保有 `requirements.txt`** - 宝塔需要这个文件安装依赖
3. **修改 `.env` 配置** - 上传后记得修改配置
4. **检查文件权限** - 确保 `www` 用户有读取权限
